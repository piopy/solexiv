# SOLEXIV

## Cosa è

Una semplice app streamlit utile a capire come e quanto spendi al mese/anno

## Come si usa

Installare le dipendenze via poetry
- `poetry install`

Lanciare l'app via terminale o tramite VSCode
- `streamlit run Homepage.py`
